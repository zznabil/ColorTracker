# Release Notes - V3.5.1

**Date:** 2026-01-03

## Highlights

- **Policeman Protocol Implementation**: Deep hardening of resource management and threading safety.
- **Stealth Mode**: Hide your GUI from OBS/Discord/Streamers with one click.
- **Instant Hotkeys**: Rebind keys on the fly without restarting the application.
- **Enhanced Feedback**: New pulsing status indicator for active tracking.
- **Precision Fixes**: Resolved Motion Engine edge cases at extreme frame rates.

# Release Notes - V3.5.0

**Date:** 2026-01-02

## Highlights

- **New "Precision Lens" Magnifier**: Circular design, 1px crosshair, dynamic data pill.
- **Motion Engine Overhaul**: Adaptive Clamping (FOV-based) and Proximity Damping (smoother tracking).
- **Architecture Refactor**: Robust "Systems First" initialization.
- **Stability**: Fixed viewport creation crash and naming inconsistencies.

### Quality
- **Modern Packaging**: Migrated to `pyproject.toml` standard.
- **Dependency Updates**: Upgraded to latest stable versions of Dear PyGui, MSS, and OpenCV.
